Приложение по просмотру и корректировке адресов.

Формы:
1. Авторизация пользователя при входе.
2. Форма "Список адресов". Колонки "Страна", "Регион", "Населенный
пункт", "Улица", "дом/корпус/квартира" (просто текстовые поля).
3. Форма "Бланк описания адреса". Для корректировки и просмотра
атрибутов адреса.
4. Отдельная форма для просмотра и ввода текстового описания адреса -
просто большое текстовое поле для ввода чего-угодно. Должна иметь
возможность быть вызвана из формы "Список адресов" и из формы "Бланк
описания адреса". По организации данных - это связанное описание по
ключам с адресом, связь одни к одному: Адрес <-> Текстовое описание.
5. Печать из формы "Список адресов" всех записей списка.

Реализовать:
- Все данные хранить в XML. Структура XML-файлов произвольная.
- Интерфейс страниц произвольный.
- Для печати использовать любой вариант.
